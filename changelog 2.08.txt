Изменения в патче 2.08


1 Новое
1.1 Добавлена реликвия "Зашифрованный Терминал", которая позволяет активировать 4-значный код и получить награду. Возможные коды могут быть получены при исследовании аномалий, расчистке планетарных блокировщиков, и от Кураторов.
1.2 Добавлен сюжет к раскопкам реликвии "Куб Связи" на ресурсе "Подземный вычислительный комплекс".
1.3 Добавлен сюжет к раскопкам реликвии "Куб Связи" на ресурсе "Торсионная катушка".
1.4 Добавлен сюжет к раскопкам реликвии "Куб Связи" на ресурсе "Галактический курорт".
1.5 Добавлена совместимость с модом "Legendary Wolds".

2 Изменено
2.1 Добавлен текст в описании зданий мода, поясняющий что такое Древние Знания (тот фиолетовый треугольник).
2.2 Изменено стартовое сообщение с параметрами настройки модификации.
2.3 Рабочие места, создаваемые модом, теперь получают бонус от соответствующих технологий и зданий (например, +10% к производству минералов шахтерами).
2.4 Время очистки непригодных планетарных залежей типа "Зараженная территория" снижено с 1000 до 500 дней.
2.5 Проведена работа по оптимизации кода, отвечающего за расстановку меток особых планет.

3 Исправлены ошибки
3.1 Исправлена ошибка, когда оружие Колосса не могло быть использовано на Уникальных и Древних мирах.
3.2 Исправлена часть английской локализации (спасибо Ник).

Также был создан Патреон модификации: https://www.patreon.com/apsr




Changes in patch 2.08


1 New
1.1 Added relic "Encrypted Terminal", which allows you to activate a 4-digit code and receive a reward. Possible codes can be obtained by researching anomalies, clearing planetary blockers, and from the Curators.
1.2 Added an new excavation text for relic "Communication Cube" on the resource "Underground Computer System".
1.3 Added an new excavation text for relic "Communication Cube" on the resource "Torsion Coil".
1.4 Added an new excavation text for relic "Communication Cube" on the resource "Galactic Resort".
1.5 Added compatibility with the "Legendary Wolds" mod.

2 Changes
2.1 Added text in the description in mod buildings, explaining what is Ancient Knowledge (that purple triangle).
2.2 Changed start message with modification settings.
2.3 Jobs from mod now receive a bonus from the corresponding technologies and buildings (for example, + 10% to the production of minerals by miners).
2.4 The cleaning time for unusable planetary deposits "Contaminated area" has been reduced from 1000 to 500 days.
2.5 Work has been done to optimize the code responsible for the placement of points of special planets.

3 Bugs fixed
3.1 Fixed a bug where Colossus weapons could not be used on Unique and Ancient worlds.
3.2 Fixed some of the English localization (thanks Nick).

Also Patreon was created for modification: https://www.patreon.com/apsr
