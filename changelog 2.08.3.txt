Изменения в патче 2.08.3


1 Новое
1.1 Добавлено специальное пояснение в описание реликвии "Зашифрованный Терминал", которая появляется если мод был включен после начала игры.
1.2 Добавлена проверка, был ли мод включен с самого начала игры. Если нет - в наступлением нового игрового года игра предложит создать особые планеты.

2 Изменено
2.1 Планетарное решение "Фокус исследований" теперь создается мгновенно.
2.2 Изменен баланс появления кодов для Терминала - уменьшен шанс для расчистки планетарных блокировщиков, увеличен шанс для кураторов после 50-100 лет.
2.3 Изменен баланс в технологиях улучшений производительности ресурсов - теперь они будут появляться позже.
2.4 Больше не приходит стартовое сообщение о планетах и не появляется специальный раздел в журнале, если при генерации выбрано "не генерировать особые планеты с ресурсами".
2.5 Появление специального раздела в журнале, который содержит информацию о найденных особых планетах, изменено - теперь он появляется не в начале игры, а после нахождения одной из особых планет.
2.6 Изменены иконки технологий для Бездны.

3 Исправлены ошибки
3.1 Поправлена неправильная работа некоторых рабочих мест.




Changes in patch 2.08.3


1 New
1.1 Added a special explanation to the description of the "Encrypted Terminal" relic, which appears if the mod was turned on after the start of the game.
1.2 Added a check if the mod was enabled from the beginning of the game. If not, the game will offer to create special planets in the beginning of the new game year.

2 Changes
2.1 Planetary Solution "Research Focus" is now created instantly.
2.2 Changed the balance of the appearance of codes for the Terminal - reduced the chance by clearing planetary blockers, increased the chance by curators after 50-100 years.
2.3 Changed the balance in technologies for improving resource productivity - now they will appear later.
2.4 A start message about planets no longer comes and a special section does not appear in the journal if "do not generate special planets with resources" is selected during generation.
2.5 The appearance of a special section in the journal, which contains information about the found special planets, has been changed - now it appears not at the beginning of the game, but after finding one of the special planets.
2.6 Changed technologies icons for the Abyss.

3 Bugs fixed
3.1 Fixed incorrect work of some jobs.
